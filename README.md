
VEMAX: Gestor de Finanzas Inteligentes

VEMAX es una plataforma intuitiva diseñada para ayudarte a tomar el control total de tus finanzas personales. Olvídate de los registros en papel; con VEMAX puedes gestionar tus ingresos y gastos de forma eficiente, visualizando tu salud financiera mediante gráficas dinámicas y tablas detalladas.

¿Qué puedes hacer con VEMAX?
Gestión de Ingresos y Gastos: Registra tus movimientos diarios en segundos.

Dashboard Visual: Entiende tus patrones de consumo a través de gráficas interactivas.
Historial Detallado: Consulta tablas organizadas de todas tus transacciones pasadas.
Balance en Tiempo Real: Visualiza tu saldo actual de forma clara y rápida.

Tecnologías Utilizadas
Backend: FastAPI (Python) para una gestión de datos rápida y robusta.
Frontend: HTML5, CSS3 y JavaScript para una experiencia de usuario fluida.
Visualización: Integración de gráficas para el análisis financiero.
Control de Versiones: Git y GitHub.
